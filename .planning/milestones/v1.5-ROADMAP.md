## Milestone Summary

**Key Decisions:**
- CSS @property for animatable beam border (conic gradient rotation)
- useScrollInView hook with continuous 0-1 progress (not binary IntersectionObserver)
- maxScale calculation to prevent mobile viewport overflow
- Plain `<img>` for dnd-icon.png (avoids Next.js Image cache issues)
- Shared AppNav component for authenticated pages (server component with logOut action)

**Additional work shipped beyond requirements:**
- FAQ accordion section (6 questions with smooth expand/collapse)
- "Built for Dungeon Masters" showcase section (campaigns + instant updates cards)
- AppNav component with breadcrumb eyebrow on campaign detail
- Hover pulse effects on all interactive cards
- Copy polish across all landing page sections

**Issues Resolved:**
- Mobile overflow from scale(1.5) on wide elements — solved with maxScale + overflow-x-clip
- Beam border bleeding through element background — solved with mask-composite: exclude
- Next.js Image caching preventing icon updates — switched to plain img tags

**Technical Debt Incurred:**
- None significant
