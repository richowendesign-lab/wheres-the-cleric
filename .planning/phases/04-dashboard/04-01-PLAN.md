**formatDateKey** (internal helper — same pattern as AvailabilityCalendar.tsx):
```ts
function formatDateKey(date: Date): string {
  return `${date.getUTCFullYear()}-${String(date.getUTCMonth() + 1).padStart(2, '0')}-${String(date.getUTCDate()).padStart(2, '0')}`
}
```

**resolvePlayerStatusOnDate** — override beats weekly, no-response when neither:
- Check for override: entry with type === 'override' whose date (parsed via Date.UTC from the stored ISO string) matches the dateKey. IMPORTANT: use `.find()` not `.filter()` — the schema guarantees at most one override per (player, date).
- If override found: return override.status as PlayerDayStatus ('free' or 'busy')
- Fall back to weekly pattern: find entry with type === 'weekly' && dayOfWeek === dow && status === 'free'
- If weekly found: return 'free'
- Else: return 'no-response'

NOTE: timeOfDay is NOT populated in this codebase (saveWeeklyPattern and toggleDateOverride do not save it). Do not use timeOfDay in aggregation.

**computeDayStatuses(playerSlots, windowStart, windowEnd)** where windowStart/End are ISO strings (already serialized):
- Return [] if either window bound is null/undefined/empty string
- Parse windowStart and windowEnd with Date.UTC from the parts (split '-' → [y, m, d])
- Iterate with a cursor from start to end (inclusive), incrementing with setUTCDate
- For each day:
  - A player has 'no-response' if slot.availabilityEntries.length === 0 (NEVER check by type — weekly-only players must not show as missing)
  - Otherwise call resolvePlayerStatusOnDate
  - Count freeCount (only 'free' statuses)
  - allFree = freeCount === playerSlots.length && playerSlots.length > 0
- Return DayAggregation[]

**computeBestDays(days)** — pure sort, no library:
```ts
export function computeBestDays(days: DayAggregation[]): DayAggregation[] {
  return [...days]
    .filter(d => d.freeCount > 0)
    .sort((a, b) => {
      if (b.freeCount !== a.freeCount) return b.freeCount - a.freeCount
      return a.date.localeCompare(b.date)  // ISO strings sort correctly
    })
    .slice(0, 5)
}
```

**formatBestDayLabel(dateKey)** — format 'YYYY-MM-DD' to 'Sat 8 Mar' style:
```ts
export function formatBestDayLabel(dateKey: string): string {
  const [y, m, d] = dateKey.split('-').map(Number)
  const date = new Date(Date.UTC(y, m - 1, d))
  return date.toLocaleDateString('en-GB', {
    weekday: 'short', day: 'numeric', month: 'short', timeZone: 'UTC',
  })
}
```
