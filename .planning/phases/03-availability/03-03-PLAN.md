    **State initialization from initialEntries:**
    - `weeklySelection: Set<string>` — derived from `initialEntries.filter(e => e.type === 'weekly')`. Each entry maps to `"{dayOfWeek}-{timeOfDay}"` (e.g. `"1-morning"`). Initialise with `new Set(weeklyEntries.map(e => \`${e.dayOfWeek}-${e.timeOfDay}\`))`.
    - `overrides: Map<string, 'free' | 'busy'>` — derived from `initialEntries.filter(e => e.type === 'override')`. Convert each entry's date to "YYYY-MM-DD" using UTC methods. `new Map(overrideEntries.map(e => { const d = new Date(e.date!); const key = \`${d.getUTCFullYear()}-${String(d.getUTCMonth()+1).padStart(2,'0')}-${String(d.getUTCDate()).padStart(2,'0')}\`; return [key, e.status as 'free' | 'busy'] }))`.
    - `saveStatus: 'idle' | 'saving' | 'saved' | 'error'` — starts `'idle'`.

    **Auto-save hooks:**
    ```typescript
    const [isPending, startTransition] = useTransition()
    const [saveStatus, setSaveStatus] = useState<SaveStatus>('idle')

    // Debounced weekly save — fires 600ms after last toggle
    const debouncedSaveWeekly = useDebouncedCallback(
      (selection: Set<string>) => {
        startTransition(async () => {
          setSaveStatus('saving')
          const entries = Array.from(selection).map(key => {
            const [day, time] = key.split('-')
            return { dayOfWeek: Number(day), timeOfDay: time }
          })
          const result = await saveWeeklyPattern(playerSlotId, entries)
          if ('error' in result) {
            setSaveStatus('error')
          } else {
            setSaveStatus('saved')
            setTimeout(() => setSaveStatus('idle'), 2000)
          }
        })
      },
      600
    )
    ```

    **Flush on unmount** (prevents lost saves on tab close):
    ```typescript
    useEffect(() => {
      return () => { debouncedSaveWeekly.flush() }
    }, [debouncedSaveWeekly])
    ```

    **Weekly toggle handler:**
    ```typescript
    function handleWeeklyChange(newSelection: Set<string>) {
      setWeeklySelection(new Set(newSelection))  // update local state immediately
      debouncedSaveWeekly(newSelection)           // persist after debounce
    }
    ```

    **Date click handler (override toggle — not debounced, fires immediately via useTransition):**
    The toggle cycle per CONTEXT.md: in-window date with no override → first click → 'busy' override; second click → 'free' override; third click → remove override (revert to weekly pattern).
    ```typescript
    function handleDateClick(dateKey: string) {
      const current = overrides.get(dateKey)
      let newStatus: 'free' | 'busy' | null
      if (current === undefined) newStatus = 'busy'
      else if (current === 'busy') newStatus = 'free'
      else newStatus = null  // remove override

      // Update local state immediately
      const newOverrides = new Map(overrides)
      if (newStatus === null) newOverrides.delete(dateKey)
      else newOverrides.set(dateKey, newStatus)
      setOverrides(newOverrides)

      // Persist via Server Action (non-debounced — each date click is discrete)
      startTransition(async () => {
        setSaveStatus('saving')
        const result = await toggleDateOverride(playerSlotId, dateKey, newStatus)
        if ('error' in result) {
          setSaveStatus('error')
          // Rollback local state on failure
          setOverrides(overrides)
        } else {
          setSaveStatus('saved')
          setTimeout(() => setSaveStatus('idle'), 2000)
        }
      })
    }
    ```

    **Save indicator sub-component (inline in same file):**
    ```typescript
    function SaveIndicator({ status }: { status: SaveStatus }) {
      return (
        <div className={`text-sm transition-opacity duration-500 ${status === 'idle' ? 'opacity-0' : 'opacity-100'} ${status === 'saved' ? 'text-green-400' : ''} ${status === 'saving' ? 'text-gray-400' : ''} ${status === 'error' ? 'text-red-400' : ''}`}>
          {status === 'saved' && 'Availability saved'}
          {status === 'saving' && 'Saving...'}
          {status === 'error' && (

                // Retry: re-trigger last save

              }}>try again</button>
            </span>
          )}
        </div>
      )
    }
    ```
