  <action>
Create the directory src/hooks/ and the file src/hooks/useInView.ts with a generic IntersectionObserver hook.

The hook signature:
```typescript
export function useInView<T extends HTMLElement = HTMLElement>(
  options: { threshold?: number; rootMargin?: string } = {}
): { ref: React.RefObject<T | null>; inView: boolean }
```

Implementation requirements:
- Use `useRef<T | null>(null)`, `useState(false)`, `useEffect`
- Create IntersectionObserver in useEffect
- On intersection: call `setInView(true)` then `observer.disconnect()` (fires once only — do NOT re-trigger on scroll back)
- Default threshold: `0.1`; default rootMargin: `'0px'`
- Cleanup: `return () => observer.disconnect()` at end of useEffect
- Dependencies array for useEffect: `[options.threshold, options.rootMargin]`
- No 'use client' directive — this is a utility module, not a component

Do NOT add 'use client' to this file. Hooks are not components.
