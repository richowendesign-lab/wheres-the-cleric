   ```tsx

   &lt;/div&gt;
   ```
   with:
   ```tsx
   &lt;div
     ref={zoomRef}
     className={[
       'transition-transform duration-500 ease-out motion-reduce:transition-none',

     ].join(' ')}
   &gt;
     &lt;PlayerDemoWidget /&gt;
   &lt;/div&gt;
   ```

CRITICAL: the zoom ref goes on a NEW inner wrapper div, NOT on the section element. The section already has ref={ref} from useInView and owns the Phase 21 fade/slide-up animation. Applying scale transform to the same element as the fade/slide animation causes jitter. The zoom wrapper is a separate div that wraps only the demo widget.
